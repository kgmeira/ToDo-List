## 📌 Sobre o Projeto

Aplicação web de lista de tarefas desenvolvida com **HTML, CSS e
JavaScript puro**, com foco em produtividade, organização e experiência
do usuário.

------------------------------------------------------------------------

## ✨ Demonstração

> Interface moderna com agenda, filtros, modo escuro e gamificação.

------------------------------------------------------------------------

## 📌 Funcionalidades

### ✅ Gerenciamento de Tarefas

-   Adicionar tarefas com descrição
-   Definir prioridade (Alta, Média, Baixa)
-   Marcar como concluída
-   Remover tarefas

### 📅 Agenda Integrada

-   Definir data
-   Filtros por Hoje / Semana / Todas
-   Destaque automático para tarefas do dia

### 🔔 Notificações

-   Alerta de tarefas do dia ao abrir o app

### 📊 Progresso

-   Barra dinâmica de conclusão

### 🌙 Modo Escuro

-   Alternância com um clique

### 🧠 Gamificação

-   Sistema de pontos por produtividade

### 💾 Persistência

-   Dados salvos com LocalStorage
